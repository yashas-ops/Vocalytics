  <behavior>
    ## Behavior Specification

    `compute_confidence(eye_contact_pct, filler_rate_per_100, wpm, speed_classification, dominant_emotion) -> ConfidenceScores`

    ### Component Scoring Rules (per D-01 through D-04)

    1. **eye_contact_score**: Maps directly from `eye_contact_pct` (0-100)
       - Score = `min(100, eye_contact_pct)` (already 0-100 scale)
       - Bounds: clamped 0-100

    2. **filler_score**: Inverse of filler word rate (per D-01 weight: 25%)
       - filler_rate_per_100 = (total_filler_count / total_words) * 100
       - `filler_rate 0 → 100`, `filler_rate >= 10 → 0`, linear in between
       - Formula: `score = max(0, 100 - filler_rate_per_100 * 10)`
       - Bounds: clamped 0-100
       - Edge case: if filler_rate_per_100 is 0, score = 100

    3. **pacing_score**: Based on WPM classification (per D-03)
       - If `speed_classification == "good"`: score = 100
       - If `speed_classification == "slow"`: linear decay from 100 at 110 WPM to 0 at 50 WPM
         - `score = max(0, (wpm - 50) / (110 - 50) * 100)`
       - If `speed_classification == "fast"`: linear decay from 100 at 160 WPM to 0 at 240 WPM
         - `score = max(0, (240 - wpm) / (240 - 160) * 100)`
       - Bounds: clamped 0-100

    4. **clarity_score**: Derived from WPM distance to ideal (135 WPM) per D-04
       - Measures how close WPM is to ideal speaking pace for clarity
       - Formula: `score = max(0, 100 - abs(wpm - 135) * 1.0)`
       - At 135 WPM → 100, at 100 WPM → 65, at 60 WPM → 25
       - Bounds: clamped 0-100

    5. **emotion_score**: Maps dominant emotion to score (per D-01 weight: 10%)
       - Mapping dictionary:
         - `"happy" → 100`, `"neutral" → 80`, `"surprise" → 70`
         - `"sad" → 40`, `"angry" → 30`, `"fear" → 20`, `"disgust" → 10`
         - `"uncertain" → 50` (fallback for unknown emotions)
       - Case-insensitive matching

    6. **composite**: Weighted sum per D-03
       - `composite = (eye_contact_score * 0.30) + (filler_score * 0.25) + (pacing_score * 0.20) + (clarity_score * 0.15) + (emotion_score * 0.10)`
       - Rounded to 1 decimal place

    7. **classification**: Thresholds per D-02
       - `composite >= 80 → "Excellent"`
       - `composite >= 60 → "Good"`
       - `composite < 60  → "Needs Improvement"`

    ### Edge cases
    - WPM of 0 → pacing_score = 0, clarity_score = 0
    - Empty/unknown emotion → use "uncertain" → score 50
    - Very high WPM (>240) → pacing_score = 0, clarity_score = 0
    - Very high filler rate (>10/100 words) → filler_score = 0
    - All scores at minimum (0) → composite = 0, classification = "Needs Improvement"
    - All scores at maximum (100) → composite = 100, classification = "Excellent"
  </behavior>

    **Report structure** (per D-05): Three sections separated by `\n\n---\n\n`:

    ### Strengths Section
    - For each of the 5 components where `component_score >= 70`:
      - Eye contact: `"✅ Strong Eye Contact — You maintained eye contact {pct}% of the time."`
      - Filler words: `"✅ Low Filler Usage — Only {count} filler words ({rate:.1f}/100 words). Your speech is clear and direct."`
      - Pacing/Clarity: `"✅ Well-Paced — {wpm} WPM is within the ideal 110-160 range. Good speaking rhythm."`
      - Emotion: `"✅ Positive Demeanor — Predominantly {emotion}, which conveys confidence and engagement."`
    - If composite >= 80: `"✅ Overall: Excellent confidence score of {composite}. Strong performance across all metrics."`
    - If no strengths: `"Keep practicing — every metric has room for improvement."`

    ### Weaknesses Section
    - For each of the 5 components where `component_score < 60`:
      - Eye contact: `"⚠️ Eye Contact Needs Work — Only {pct}% eye contact. Try looking directly at the camera lens."`
      - Filler words: `"⚠️ Frequent Filler Words — {rate:.1f} per 100 words ({count} total). Reducing fillers will make you sound more polished."`
      - Pacing: `"⚠️ Speaking Pace is {classification.upper()} — {wpm} WPM. Target the 110-160 range for best clarity."`
      - Clarity: `"⚠️ Speaking Clarity Affected by Pace — {wpm} WPM is outside the ideal range. Practice modulating your speed."`
      - Emotion: `"⚠️ Facial Expressions Appear {emotion.upper()} — Try to project more confidence through your expressions."`
    - If no weaknesses: No weaknesses section added (per D-05, only show what's relevant)

    ### Improvement Tips Section
    - Generate actionable tips based on which components scored low (per D-06):
      - `eye_contact_score < 70`: "- **Camera Practice**: Place a sticky note next to your camera lens. Every time you speak, glance at it as a reminder to make 'eye contact' with the camera."
      - `filler_score < 70`: "- **Pause Instead of Filler**: When you feel 'um' or 'uh' coming, pause silently instead. Record yourself and count filler words — awareness reduces them by 30-50% with practice."
      - `pacing_score < 70 and speed_classification == "fast"`: "- **Slow Down**: Practice pausing between key points. Try the 'one breath per sentence' technique to naturally regulate your pace."
      - `pacing_score < 70 and speed_classification == "slow"`: "- **Pick Up the Pace**: Practice with a timer — aim to cover key points within 2-minute blocks. Read passages aloud to build fluency."
      - `clarity_score < 80`: "- **Clarity Drill**: Record yourself reading a short passage at 120-150 WPM. Replay and check if every word is clear. Adjust until your natural pace lands in this range."
      - `emotion_score < 60`: "- **Facial Engagement**: Practice speaking with slight smile (for neutral/positive topics). Record yourself and check if your expression matches your message."
    - Each tip is one bullet point, prefixed with `- **Title**: Description.` (per D-06: concrete, per-metric)

    All templates use Python f-strings/%-formatting (no spaCy needed per D-07).

    Return a single string with `\n\n---\n\n` section separators. The result goes into `st.expander` on the Dashboard (per D-08), so use markdown formatting but keep it compact.
