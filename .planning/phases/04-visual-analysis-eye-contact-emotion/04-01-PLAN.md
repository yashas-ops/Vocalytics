### Existing Models (from modules/models.py)

```python
class FrameEyeContact(BaseModel):
    """Eye contact result for a single video frame."""
    frame_index: int
    has_face: bool
    looking_at_camera: bool
    confidence: float

class EyeContactResult(BaseModel):
    """Aggregated eye contact analysis across sampled frames."""
    contact_percentage: float
    total_frames: int
    contact_frames: int
    frame_results: List[FrameEyeContact]

class EmotionScores(BaseModel):
    """Emotion probability distribution."""
    angry: float = 0.0
    disgust: float = 0.0
    fear: float = 0.0
    happy: float = 0.0
    sad: float = 0.0
    surprise: float = 0.0
    neutral: float = 0.0

class EmotionResult(BaseModel):
    """Dominant emotion and per-frame analysis results."""
    dominant_emotion: str
    emotion_distribution: Dict[str, float]
    frames_analyzed: int
```

### Key Decisions from CONTEXT.md

| ID | Decision |
|----|----------|
| D-04 | Time-based keyframe sampling — 1 frame every N seconds, soft cap 20 frames |
| D-05 | Sample interval = max(1, total_duration / 20) seconds |
| D-06 | Frames downscaled to 640x480 before DeepFace |
| D-07 | MediaPipe Face Mesh legacy API (`mp.solutions.face_mesh`) |
| D-08 | Eye contact via PnP head pose (yaw ±15°, pitch ±10°) |
| D-09 | Convert BGR→RGB before MediaPipe |
| D-10 | Store in existing EyeContactResult, FrameEyeContact models |
| D-11 | DeepFace: detector_backend='opencv', enforce_detection=False, actions=['emotion'] |
| D-12 | Only report emotion if DeepFace confidence > 50% |
| D-13 | Soft voting across keyframes for final emotion distribution |
| D-14 | Store in existing EmotionResult, EmotionScores models |
| D-15 | If MediaPipe found face, crop to ROI before DeepFace |
| D-18 | If MediaPipe fails on a frame, mark has_face=False, looking_at_camera=False |

### 3D Face Model for PnP Head Pose

The 6 key MediaPipe landmark indices used for PnP:

| Landmark | MediaPipe Index |
|----------|----------------|
| Nose tip | 1 |
| Chin | 199 |
| Left eye left corner | 33 |
| Right eye right corner | 263 |
| Left mouth corner | 61 |
| Right mouth corner | 291 |

    ### GREEN PHASE: Implement module

    Create `modules/visual_analysis.py` with the following functions:

    **1. `extract_keyframes(video_path: str, max_frames: int = 20) -> Tuple[List[np.ndarray], List[int]]`**
    - Open video with `cv2.VideoCapture(video_path)`
    - Get total_frames, fps
    - Calculate interval: `max(1, total_frames // max_frames)` (per D-05)
    - Loop at interval, seek with `cap.set(cv2.CAP_PROP_POS_FRAMES, i)`, read frame
    - Downscale each frame to 640x480 with `cv2.resize(frame, (640, 480))` (per D-06)
    - Collect frames and their original frame indices
    - Stop when max_frames reached or video ends
    - Release capture and return `(frames, frame_indices)`
    - Handle errors: if video_path doesn't exist or can't open, return `([], [])`

    **2. `_compute_head_pose(landmarks, image_shape: Tuple) -> Tuple[Optional[float], Optional[float], Optional[float]]`**
    - Extract 6 key 2D landmark points (indices 1, 199, 33, 263, 61, 291) from MediaPipe landmarks
    - Convert normalized coords to pixel coords: `x * w, y * h`
    - Define 3D model points using the canonical face model coordinates from the Context section
    - Set up camera matrix: focal_length = image_width, center = (w/2, h/2)
    - Call `cv2.solvePnP(model_points_3d, image_points_2d, camera_matrix, dist_coeffs, flags=cv2.SOLVEPNP_ITERATIVE)`
    - Convert rotation vector to matrix with `cv2.Rodrigues()`
    - Extract Euler angles:
      - pitch = `-arcsin(R[2,1]) * 180/π`
      - yaw = `arctan2(R[2,0], R[2,2]) * 180/π`
      - roll = `arctan2(R[0,1], R[1,1]) * 180/π`
    - Return `(yaw, pitch, roll)` — or `(None, None, None)` if solvePnP fails

    **3. `check_eye_contact(yaw: Optional[float], pitch: Optional[float], roll: Optional[float], yaw_threshold: float = 15.0, pitch_threshold: float = 10.0) -> bool`**
    - If yaw is None: return False
    - Return `abs(yaw) <= yaw_threshold and abs(pitch) <= pitch_threshold`
    - Note: roll is not used for eye contact threshold (per D-08)

    **4. `analyze_eye_contact(frames: List[np.ndarray], face_mesh, frame_indices: Optional[List[int]] = None, image_shape: Optional[Tuple] = None) -> EyeContactResult`**
    - Initialize empty frame_results list, contact_count = 0
    - For each frame:
      - Convert BGR→RGB (per D-09): `rgb = cv2.cvtColor(frame, cv2.COLOR_BGR2RGB)`
      - Process with face_mesh: `results = face_mesh.process(rgb)`
      - Determine looking:
        - If `results.multi_face_landmarks`: compute head pose from `results.multi_face_landmarks[0].landmark`, check eye contact
        - If no face: `has_face=False, looking_at_camera=False, confidence=0.0`
      - Store per-frame confidence based on how centered the head pose is (1.0 - normalized deviation from center)
      - Append FrameEyeContact to result list
    - Return EyeContactResult with aggregated stats
    - Handle per D-18: If MediaPipe fails on individual frames, mark as has_face=False, looking_at_camera=False

    **5. `_aggregate_emotions(deepface_results: List[Dict]) -> Tuple[str, Dict[str, float]]`**
    - Per D-12: apply 50% confidence threshold — exclude frames where dominant emotion confidence ≤ 50%
    - Per D-13: soft voting — count how many remaining frames had each dominant emotion
    - Calculate frequency distribution: count_per_emotion / total_valid_frames
    - Dominant emotion = most frequent across valid frames (tie-break: alphabetically first)
    - Note: this function processes already-analyzed DeepFace results — it does NOT call DeepFace itself

    **6. `analyze_emotions(frames: List[np.ndarray], face_mesh, frame_indices: Optional[List[int]] = None) -> EmotionResult`**
    - For each frame:
      - Per D-15: if MediaPipe found a face, crop to face ROI before DeepFace to improve speed
      - Call DeepFace: `DeepFace.analyze(frame, actions=['emotion'], detector_backend='opencv', enforce_detection=False)` (per D-11)
      - Per D-12: Extract emotion confidence from result
      - Per D-17: if DeepFace fails on individual frame, skip and continue
    - Collect results from all successfully analyzed frames
    - Call `_aggregate_emotions()` to produce final distribution
    - Return EmotionResult

    **7. `analyze_visual(video_path: str) -> Tuple[EyeContactResult, EmotionResult]`**
    - Integration function that:
      1. Loads MediaPipe Face Mesh: `face_mesh = load_mediapipe_face_mesh()`
      2. Triggers DeepFace download: `load_deepface_model()`
      3. Extracts keyframes: `frames, indices = extract_keyframes(video_path, max_frames=20)`
      4. Analyzes eye contact: `eye_result = analyze_eye_contact(frames, face_mesh, indices)`
      5. Analyzes emotions: `emotion_result = analyze_emotions(frames, face_mesh, indices)`
      6. Returns `(eye_result, emotion_result)`
