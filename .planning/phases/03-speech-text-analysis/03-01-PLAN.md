```python
@st.cache_resource
def load_spacy_model() -> "spacy.Language":
    """Load spaCy English model for NLP analysis."""
    import spacy
    try:
        return spacy.load("en_core_web_sm")
    except OSError:
        import subprocess
        subprocess.run(
            ["python", "-m", "spacy", "download", "en_core_web_sm"],
            check=True
        )
        return spacy.load("en_core_web_sm")
```

### Filler Words to Detect

| Word | Detection Method |
|------|-----------------|
| um | Direct case-insensitive match |
| uh | Direct case-insensitive match |
| like | Match UNLESS spaCy POS is VB/VBP/VBZ/VBD (verb forms) |
| basically | Direct case-insensitive match |
| literally | Direct case-insensitive match |
| you know | Bigram match: "you" + "know" adjacent tokens |

### WPM Classification

| Range | Classification |
|-------|---------------|
| < 110 | slow |
| 110–160 | good |
| > 160 | fast |

    **`analyze_filler_words(text: str, nlp) -> List[FillerWordCount]`:**
    - Tokenize with spaCy (`nlp(text)`)
    - Filter to only alphabetic tokens (`token.is_alpha`)
    - Lowercase each token for matching
    - Initialize a dict for all 6 filler words with count 0
    - Iterate tokens:
      - If `token.lower_ in ("um", "uh")` → increment
      - If `token.lower_ == "like"` and `token.pos_` is NOT in `("VERB",)` — use `token.pos_` which returns "VERB" for VB/VBP/VBZ/VBD — actually, check `token.tag_` for fine-grained tags: "VB", "VBP", "VBZ", "VBD". Wait, the requirement says "like as verb" — so I should check `token.pos_ == "VERB"` which covers all verb forms.
      - Hmm, actually `token.pos_ == "VERB"` would also cover VBG (gerund) like "liking". But "I'm liking this" uses "like" as a verb too, which shouldn't be filler. And "It's likeable" — "likeable" is ADJ, not a problem since we're checking token.lower_ == "like".
      
      Using `token.pos_ != "VERB"` is simpler and more correct — it excludes ALL verb uses of "like" including gerunds.
      
      - If `token.lower_ == "basically"` → increment
      - If `token.lower_ == "literally"` → increment
      - If `token.lower_ == "you"` and `i + 1 < len(doc)` and `doc[i + 1].lower_ == "know"` → increment "you know", skip next token
    - Filter out words with count 0 from results
    - Return sorted list (descending count, alphabetically for ties)

    **`calculate_wpm(text: str, duration_sec: float) -> tuple[float, str]`:**
    - Tokenize with spaCy
    - Count alphabetic tokens only (`token.is_alpha`)
    - duration_min = duration_sec / 60
    - If duration_min == 0 or total_words == 0: return (0.0, "slow")
    - wpm = total_words / duration_min
    - Classify: <110 → "slow", 110-160 → "good", >160 → "fast"
    - Return (wpm, classification)
