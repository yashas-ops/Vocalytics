### Pipeline Logic (replace `if uploaded_file is not None:` block)

When a file is uploaded (`uploaded_file is not None`):

1. **Initial check:** Validate extension via `file_manager.allowed_file(uploaded_file.name)`. If not allowed, show `st.error(f"Unsupported format: {ext}. Use mp4, mov, or avi.")` and return.

2. **Add an "Analyze" button** — don't auto-start pipeline on upload. Show:
   ```
   st.success(f"📁 **{uploaded_file.name}** ({size:.1f} MB) ready for analysis")
   if st.button("🚀 Analyze Interview", type="primary", use_container_width=True):
       # Run pipeline
   ```
   This gives the user control over when processing starts (important for large files).

3. **Inside the button handler, use `st.progress()` and `st.status()`** for pipeline feedback:

   ```python
   progress_bar = st.progress(0, text="Starting pipeline...")
   with st.status("Processing interview...", expanded=True) as status:
       # Step 1: Save video
       progress_bar.progress(10, text="Saving video...")
       st.write("📁 Saving video to uploads/...")
       video_path = save_upload(uploaded_file)
       interview_id = insert_interview(video_path)
       st.write(f"✅ Video saved ({get_file_size_mb(video_path):.1f} MB)")
       st.write(f"📋 Interview ID: `{interview_id}`")

       # Step 2: Extract audio
       progress_bar.progress(30, text="Extracting audio...")
       st.write("🎵 Extracting audio (16kHz mono WAV)...")
       audio_result = extract_audio(video_path)
       if not audio_result.success:
           st.error("❌ Audio extraction failed. Ensure ffmpeg is installed and on your PATH.")
           status.update(label="Audio extraction failed", state="error")
           return
       st.write(f"✅ Audio extracted: {audio_result.duration_sec:.0f}s duration")
       st.write(f"   Saved to: `{audio_result.audio_path}`")

       # Step 3: Transcribe
       progress_bar.progress(60, text="Transcribing (faster-whisper)...")
       st.write("📝 Transcribing speech to text...")
       st.caption("This may take a moment depending on video length. "
                   f"Expected: ~{audio_result.duration_sec:.0f}s of audio.")
       transcript = transcribe_audio(audio_result.audio_path)
       st.write(f"✅ Transcription complete: {len(transcript.segments)} segments")
       st.write(f"   Model: `{transcript.model_used}`")
       st.write(f"   Total duration: {transcript.duration_sec:.1f}s")

       # Complete
       progress_bar.progress(100, text="Pipeline complete!")
       status.update(label="Analysis complete", state="complete", expanded=False)
   ```

4. **After pipeline completes** (outside the `with st.status()` block but still inside the button handler if needed — actually keep inside the status block for a clean UX, or use two status blocks if separation is cleaner):

   Keep the display inside the status. After the status block closes, show a success message and the transcript results:

   ```python
   # Display results after pipeline
   st.markdown("---")
   st.markdown("### 📝 Transcript")
   st.markdown(f"**{len(transcript.segments)} segments** across {transcript.duration_sec:.1f}s")

   # Show segments in an expandable container
   for i, seg in enumerate(transcript.segments):
       with st.expander(f"Segment {i+1}: {seg.start:.1f}s – {seg.end:.1f}s"):
           st.write(seg.text)

   # Also show full text in a text area (collapsed by default)
   with st.expander("📄 Full Transcript Text", expanded=False):
       st.text_area("Full transcript", transcript.full_text, height=200, disabled=True)

   st.success("✅ Analysis complete! Visit the **Dashboard** page to view detailed results.")
   ```

5. **Sidebar note:** Add a small indicator in the sidebar when results exist — show `st.session_state.get("last_interview_id", None)`. If set, show "Latest: Interview {id}" in the sidebar.

### Error handling
- If ffmpeg is not installed: `subprocess.CalledProcessError` or `FileNotFoundError` will propagate. The status block's `state="error"` should catch it. Wrap Step 2 in try/except to be safe:
  ```python
  try:
      audio_result = extract_audio(video_path)
  except FileNotFoundError as e:
      st.error(f"❌ {str(e)}")
      status.update(label="Pipeline failed: ffmpeg missing", state="error")
      return
  except Exception as e:
      st.error(f"❌ Audio extraction error: {str(e)}")
      status.update(label="Pipeline failed", state="error")
      return
  ```
- If faster-whisper fails: Similarly wrap Step 3 in try/except with a user-friendly error message.
- Store the transcript result in `st.session_state` so it survives reruns: `st.session_state.last_transcript = transcript` and `st.session_state.last_interview_id = interview_id`.
